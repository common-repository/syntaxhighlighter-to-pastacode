Convert your SyntaxHighlighter Evolved's shortcodes into Pastacode ones.

== Description ==

The only use of this plugin is to convert all [SyntaxHighlighter Evolved](https://fr.wordpress.org/plugins/syntaxhighlighter/)'s shortcodes into [Pastacode](https://fr.wordpress.org/plugins/pastacode/) ones.

This migration tool has been designed to convert a variety of shortcodes through a large number of articles and pages, using a ajax crawl so you can run it on large websites.

A report is generated automatically, allowing you to inspect the modified contents and revert changes in case of broken tags.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Launch tags conversion in submenu Tools > Migrate to Pastacode
